How To Run:
basically just go to main.py and run the program,
or if you want to run the test_blackjack.py just go there and hit run and it should tell if everything is working right.

what the proram does:
    the program to sum it up allows the person who runs it to play a simple game of blackjack but if further explanation is needed
    card.py basically is what makes the card class and suit like it makes a king a king, and main.py basically makes the game run
    it takes the stuff from game.py and creates the tkinter window and allows the game to run
    the game.py has the gui or (graphical user interface) button interactions, turns, basically everything youd expect from a blackjack game
    deck.py handles the random aspect of drawing random cards out of a standard 52 card deck 
    hand.py basically handles what the player or dealer has
