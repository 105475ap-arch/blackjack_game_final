BLACKJACK FINAL PROJECT
by:Alex Huffman
Files Included:
- main.py
    Starts the program

- game.py
    Handles the GUI and and game rules.

- card.py
    Contains the Card class

- deck.py
    Contains the Deck class

- hand.py
    Contains the Hand class

- test_blackjack.py
    Runs tests for cards, decks, and hand values

Features:
    Hit and Stand system
     Dealer AI
     Hidden dealer card
     Blackjack detection
     win streak counter
     Chip tracking
     GUI made with Tkinter

How To Run:
basically just go to main.py and run the program,
or if you want to run the test_blackjack.py just go there and hit run and it should tell if everything is working right.
